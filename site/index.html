Methodology

Every battle is captured using the browser-trace CDP skill from browserbase/skills. A read-only Chrome DevTools Protocol client attaches to the browser session and records the full network firehose to NDJSON. This gives us the exact wire-level timestamp when the /vote XHR fires, the /reveal response body containing model names, and SSE stream timestamps for TTFT measurement — none of which are accessible from the DOM.

Shadow win rate weights votes by reading time: fast votes (<5s) count 1×, slow votes (>15s) count 3×. Rank delta = Arena rank minus Shadow rank — positive means the model is underrated by fast voters, negative means overrated.
